# HSE Labs

Подборка лабораторных работ по математике и программированию на Python.

В репозитории собраны Jupyter Notebook-работы, выполненные в рамках учебных курсов ВШЭ.  
Основной фокус — применение математических методов на практике: моделирование, численные вычисления, визуализация и анализ результатов.

---

## Содержание

### Linear Regression

Ноутбук: homework-practice-02-linregr-Поздеева.ipynb

Лабораторная работа по реализации и анализу линейной регрессии.

Что сделано:

- реализация линейной регрессии;
- работа с полиномиальными признаками;
- использование L2-регуляризации;
- онлайн-обновление параметров модели;
- анализ поведения модели;
- визуализация результатов.

Использованные инструменты:

- Python
- NumPy
- Matplotlib

---

### Secret Sharing

Ноутбук: Лабораторная_работа_2_Часть_1_МА1_ПМИ_Поздеева_Анна_Николаевна_Вариант_174.ipynb

Лабораторная работа по моделированию криптографической схемы разделения секрета.

Что сделано:

- моделирование схемы разделения секрета;
- восстановление секрета через интерполяцию;
- работа с полиномами и функциями;
- исследование устойчивости схемы;
- анализ влияния параметров;
- визуализация и интерпретация результатов.

Использованные инструменты:

- Python
- NumPy
- SciPy
- SymPy
- Matplotlib
